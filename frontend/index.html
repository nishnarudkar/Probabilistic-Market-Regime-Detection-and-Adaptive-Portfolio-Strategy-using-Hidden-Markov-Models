Advanced ML: Market Regime Monitor

Hidden Markov Model (HMM) automated risk management system.

Built by Nishant Narudkar, Aamir Sarang & Saksham Shukla

User Guide

CSV Upload tab — Upload your processed market data CSV and click "Analyze Market Data". The model will classify every row into a regime and show the current signal for the latest date. Use the "Days to display" input to control how many recent trading days appear on the chart.

Manual Input tab — Enter a single set of market conditions and click "Check Market Signal" to get an instant regime prediction. Use "Load Default Values" to pre-fill typical calm market values for experimentation.

What-If Explorer — Appears after you submit the manual form. Drag the sliders to tweak values and see the signal update in real time without re-submitting. Useful for finding the boundary where the market flips from safe to risky.

What each feature means
  • Returns — The daily percentage change in price (e.g. 0.012 = +1.2%, -0.05 = -5%). Negative values indicate a falling market.
  • Volatility — Rolling standard deviation of returns over ~20 days. Low values (~0.005) = calm, high values (~0.04+) = turbulent.
  • RSI — Relative Strength Index (0–100). Below 30 = oversold/bearish, above 70 = overbought/bullish, ~50 = neutral.
  • Momentum — 10-day price change in points (not percentage). Positive = uptrend, negative = downtrend. Typical range in this dataset: -300 to +300.
  • VIX — CBOE Volatility Index, the market's "fear gauge". Below 15 = calm, 15–25 = normal, 25–35 = elevated stress, 35+ = crisis.
